Description

Weather App JS is a simple application that allows users to check the current weather conditions.

Installation

Follow these steps to install and run the app:

git clone https://github.com/Salma-NB/weatherApplication-.git
cd your-weather-app
open index.html

Usage

Simply enter the city name and get the current weather information.
